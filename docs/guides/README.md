# Rusk Guides（指南）

本目录收录面向使用者/嵌入者的入门与迁移指南（中文）。

建议阅读顺序：

1. `quick_start.md`：快速上手，跑通示例与常用命令
2. `concepts_and_syntax.md`：语言概念与语法总览（表达式、类型、接口、effects、core 库）
3. 迁移视角：
   - `for_typescript_developers.md`
   - `for_python_developers.md`
4. `host_integration.md`：把 Rusk 嵌入到你的应用里（宿主函数、外部化 effects、ABI、sysroot）

更深入的参考资料：

- 语言规范：`RUSK_SPEC.md`
- MIR 规范：`MIR_SPEC.md`
- 字节码规范：`BYTECODE_SPEC.md`
- 完整嵌入 VM 文档：`docs/embedding-vm.zh.md`
